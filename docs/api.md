# REST API

GET /

GET /health

GET /portfolio/all

Returns all portfolios.

GET /run/{portfolio_id}

Runs complete AI workflow.

GET /analytics

Returns decision analytics.

GET /backtest

Returns strategy comparison.

GET /simulation/{portfolio_id}

Monte Carlo simulation.

GET /recommend/{portfolio_id}

AI recommendations.

GET /alerts/{portfolio_id}

Portfolio alerts.

GET /timeline/{portfolio_id}

Execution timeline.

GET /agent-metrics

Agent statistics.

GET /report/{portfolio_id}

Download PDF report.
